Nel file in ./src/main/resources/crea_tabella.txt c'è il codice SQL per la creazione della tabella.

Nel file config.txt sono presenti i parametri per la connessione al DB separati da ";", i valori
dei parametri nel file sono da sostituire con quelli corretti per la connessione al vostro database,
sono inseriti nel seguente ordine:

driverjdbc;ip;porta;username;password;

(personalmente ho testato il programma su un database MS SQLServer).

Dopo aver impostato i parametri correttamente il programma è pronto per funzionare.
